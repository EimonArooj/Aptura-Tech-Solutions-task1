# Titanic Dataset - Exploratory Data Analysis (EDA)

## Overview
This project performs **Exploratory Data Analysis (EDA)** on the Titanic dataset to understand passenger characteristics, identify data patterns, and determine the factors influencing survival using statistical analysis and visualizations.

## Project Workflow
- Data Loading and Inspection
- Missing Value Analysis and Handling
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Identification of Top 3 Influential Features
- Scalability Analysis for Large Datasets

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis
- Dataset structure and summary statistics
- Missing value treatment
- Distribution of Age and Fare
- Survival analysis by Gender and Passenger Class
- Boxplots for Fare distribution
- Correlation matrix and heatmap
- Identification of the top three features affecting passenger survival using absolute correlation values

## Scalability Discussion
The project also discusses how the workflow can be redesigned to efficiently process datasets with **10 million+ records** using:
- Dask / PySpark
- Parallel Processing
- Chunk-based Data Processing
- Parquet Storage
- Sampling for Visualization
